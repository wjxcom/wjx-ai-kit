2. var、let 和 const 的区别是什么？[多选题]
var 有函数作用域，let 和 const 有块作用域
const 声明后不能重新赋值
let 声明的变量会被提升
var 可以重复声明同名变量

"null"
"undefined"
"object"
"boolean"
